# Neon Dance Instalation

## Setup

```bash
- Copy this `ğŸ“‚nuxt-app` directory into your project
- Open `ğŸ“‚nuxt-app` directory via `cd nuxt-app` in terminal of your code editor
- Run this commands `npm install` ,  `yarn install` , `pnpm install --shamefully-hoist`
```

## Development Server

Start the development server on http://localhost:3000

```bash
To start dev server run this command: `npm run dev`
```

## Production

Build the application for production:

```bash
npm run build
```

Locally preview production build:

```bash
npm run preview
```

## Deploy

Generate static files to deploy on hosting

```bash
npm run generate
```

In the `ğŸ“‚output` directory will be generated public folder, all the files in the public folder, you can transfer to the hosting, then the site will work.
